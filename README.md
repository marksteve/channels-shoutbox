# channels-shoutbox

Shoutbox demo of Django Channels

Presented in [this meetup](http://www.meetup.com/pythonph/events/231926874/)

## Run

```bash
virtualenv venv
source venv/bin/activate
python setup.py develop
./manage.py migrate
./manage.py runserver
```
